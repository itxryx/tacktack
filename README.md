# tacktack

ローカル完結型のTUIタスク管理＆タイムトラッキングツール。Go製、SQLite保存。

## 特徴

- **タスク管理** — 優先度・締切・タグ（プロジェクト/コンテキスト）付きでタスクを管理
- **タイムトラッキング** — タスクごとに計測開始/停止、累積時間を表示
- **タイムライン** — 1日の作業を30分スロットで可視化、日付をまたいで閲覧可能
- **統計** — タグ別・期間別（今日/今週/今月/半年/1年）の計測時間をバーグラフで表示
- **todo.txt ライクな表示** — 一覧は `(A) タスク名 +project @context due:2025-12-31` 形式で表示

## インストール

```bash
git clone https://github.com/itxryx/tacktack
cd tacktack
make install   # ~/go/bin/tacktack にインストール
```

または手動ビルド:

```bash
make build     # ./bin/tacktack を生成
./bin/tacktack
```

**依存**: Go 1.25+、CGO（SQLite3 のため）

## データ保存先

XDG Base Directory 仕様に従い `~/.local/share/tacktack/tacktack.db` に保存される。
`$XDG_DATA_HOME` が設定されている場合は `$XDG_DATA_HOME/tacktack/tacktack.db`。

## タスク入力フォーム

`i` または `a` キーで開くフォームで各フィールドを個別に入力する。

| フィールド | 入力例               | 説明                                                             |
| ---------- | -------------------- | ---------------------------------------------------------------- |
| タイトル   | `メールを確認`       | 必須                                                             |
| 優先度     | `A`                  | `A`〜`Z` の1文字、省略可                                         |
| 締切日     | `2025-12-31`         | `YYYY-MM-DD`、`today`、`tomorrow`、`mon`〜`sun`、`next fri` など |
| タグ       | `+` / `@` キーで選択 | プロジェクト（+）・コンテキスト（@）を別画面で選択               |

## キーバインド

### タスク一覧

| キー           | 操作                         |
| -------------- | ---------------------------- |
| `j` / `k`      | カーソル移動                 |
| `i` / `a`      | タスク新規入力               |
| `e`            | 詳細編集                     |
| `x`            | 完了/未完了 切替             |
| `Space`        | タイムトラッキング 開始/停止 |
| `d` / `BS`     | タスク削除（確認あり）       |
| `Tab`          | タイムラインビューへ         |
| `q` / `Ctrl+C` | 終了                         |

### タイムラインビュー

| キー      | 操作         |
| --------- | ------------ |
| `h` / `l` | 前日 / 翌日  |
| `j` / `k` | スクロール   |
| `t`       | 今日に戻る   |
| `Tab`     | 統計ビューへ |
| `Esc`     | タスク一覧へ |

### 統計ビュー

| キー          | 操作                                |
| ------------- | ----------------------------------- |
| `h` / `l`     | 期間切替（今日/今週/今月/半年/1年） |
| `j` / `k`     | 異常セッション一覧を移動            |
| `e`           | 異常セッションのタスクを詳細編集    |
| `Tab` / `Esc` | タスク一覧へ                        |

## ビュー切替

```
タスク一覧 → (Tab) → タイムライン → (Tab) → 統計 → (Tab) → タスク一覧
```
